# SAMD Studio — Portfolio PWA

A Progressive Web App portfolio built with pure HTML/CSS/JS.

## 🚀 Deploy to GitHub Pages

1. Upload all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source branch to `main` → Save
4. Your site will be live at: `https://username.github.io/repo-name`

## 📱 Install as App

Once deployed, open the URL in Chrome/Safari and tap **"Add to Home Screen"** to install as a native-like app.

## 📁 Files

| File | Description |
|------|-------------|
| `index.html` | Main portfolio page (all assets embedded) |
| `manifest.json` | PWA manifest — app name, icons, theme |
| `sw.js` | Service Worker — enables offline mode |
| `icon.jpg` | App icon (used on home screen) |

## ✨ Features

- 🌙 Dark / Light mode with localStorage persistence
- 🖼️ Masonry gallery with filter tabs
- 🔍 Lightbox with keyboard navigation
- 📱 Installable PWA (works offline)
- ⚡ Zero dependencies — pure HTML/CSS/JS
